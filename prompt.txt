Build a complete Smart India Hackathon (SIH) level AI-powered web application named "AirGuard AI – Air Quality Prediction & Smart Health Advisory System".

Objective:
Develop a production-ready full-stack application that predicts Air Quality Index (AQI) using Machine Learning, visualizes pollution trends, identifies pollution hotspots, and provides personalized health advisories for citizens and government authorities.

Technology Stack:
• Frontend: React.js + TypeScript + Vite
• UI: Tailwind CSS + Shadcn/UI + Framer Motion
• Backend: Python FastAPI
• Machine Learning: Scikit-learn, XGBoost, TensorFlow (LSTM), Pandas, NumPy
• Database: PostgreSQL
• Authentication: JWT
• Maps: Leaflet or Google Maps
• Charts: Recharts
• Notifications: Email and Push Notifications
• Deployment Ready: Docker

The application should have a modern, premium, responsive, and SIH-winning UI with smooth animations, glassmorphism, gradient cards, dark/light mode, beautiful dashboards, and mobile-first responsive design.
